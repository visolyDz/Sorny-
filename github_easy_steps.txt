أسهل خطوات لرفع وتشغيل موقعك على GitHub – للمبتدئين (من الهاتف)

1. إنشاء مستودع جديد

1.  افتح موقع GitHub.
2.  اضغط على زر + في الأعلى.
3.  اختر New repository.
4.  اكتب اسم المشروع.
5.  اضغط Create repository.

2. رفع ملفات الموقع

1.  انزل للأسفل إلى قسم Upload files.
2.  اضغط عليه.
3.  اختر الملفات من هاتفك (HTML – CSS – JS – صور…).
4.  اضغط Commit changes لحفظها.

3. تشغيل الموقع (GitHub Pages)

1.  من داخل Repo اضغط Settings.
2.  انزل حتى تجد Pages.
3.  في Source اختر Main ثم /root.
4.  اضغط Save.
5.  يظهر لك رابط موقعك بعد دقيقة.

4. فتح موقعك

بعد تفعيل GitHub Pages سيعطيك رابط مثل:

https://username.github.io/repository-name

افتح الرابط وسيعمل موقعك مباشرة على الإنترنت.
